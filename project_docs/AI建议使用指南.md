# AI 建议功能 - 快速使用指南

## 🎯 功能简介

AI 建议功能使用 **DeepSeek AI** 分析您的衣橱数据，提供个性化的管理建议。

## 🚀 如何使用

### 1. 访问建议页面

在应用中点击 **"智能建议"** 菜单，或直接访问：
```
http://localhost:3000/dashboard/recommendations
```

### 2. 查看 AI 建议

页面会显示：
- 🤖 AI 智能分析模式指示器
- 3 条个性化建议
- 每条建议包含：
  - 标题和优先级
  - 详细描述
  - 分析理由
  - 可操作按钮（如有）

### 3. 切换衣橱

使用页面右上角的下拉菜单，可以切换不同的衣橱查看建议。

## 💡 建议类型

### 🛍️ 购物建议
- 分析衣橱结构，提供合理的购买建议
- 示例："准备春季外套"

### 👔 搭配建议
- 根据搭配数据，推荐穿搭组合
- 示例："创建更多搭配组合"

### 📊 使用建议
- 分析使用频率，提升利用率
- 示例："激活闲置运动装"

### 🧹 整理建议
- 提醒需要整理的衣物
- 示例："控制购买频率"

### 🎨 风格建议
- 根据品牌和风格，提供多样化建议
- 示例："尝试更多品牌"

## 🎨 优先级说明

| 优先级 | 颜色 | 含义 |
|--------|------|------|
| **高** | 🔴 红色 | 需要立即关注的问题 |
| **中** | 🟡 黄色 | 建议尽快处理 |
| **低** | 🔵 蓝色 | 可选的优化建议 |

## 🧠 AI 如何分析

DeepSeek AI 会分析以下数据：

### 1. 衣橱数据
- 衣物数量、分类、品牌
- 价格、使用频率
- 购买时间、状态

### 2. 季节信息
- 当前季节（春/夏/秋/冬）
- 即将到来的季节
- 季节性衣物准备情况

### 3. 用户偏好
- 风格偏好（基于标签）
- 喜欢的颜色
- 喜欢的品牌
- 预算水平

### 4. 购买习惯
- 最近30天购买数量
- 购买频率（高/中/低）

### 5. 使用情况
- 衣物利用率
- 闲置衣物识别
- 高价衣物性价比

## 📊 建议示例

### 示例 1：季节准备建议

```

春季即将到来，但您只有2件轻薄外套。建议购买1-2件
春季夹克或风衣，适合早春穿搭。

💡 分析理由：当前季节为冬季，即将进入春季。数据
显示您的春季外套数量不足，且您的购买频率较高，
适合提前准备。

[浏览春季外套推荐]
```

### 示例 2：使用频率建议

```
📊 激活闲置运动装 [中优先级]

您有5件运动服超过30天未穿，与您的运动风格偏好不
符。建议本周尝试搭配运动装出门。

💡 分析理由：用户风格偏好包含运动风，但运动类衣
物利用率仅为35%，存在明显的使用与偏好不匹配。

[查看运动装建议]
```

### 示例 3：购买控制建议

```
🧹 控制购买频率 [低优先级]

您最近30天购买了5件衣物，购买频率较高。建议先充
分利用现有衣物，暂缓新购。

💡 分析理由：购买频率为高频，但衣橱利用率仅为
58.3%，说明现有衣物未被充分使用。
```

## 🔄 刷新建议

- 下拉页面即可刷新建议
- 建议会缓存 1 小时，避免频繁调用
- 添加新衣物或搭配后，建议等待缓存过期或手动刷新

## ⚙️ 工作模式

### 模式 1：DeepSeek AI（默认）
- ✅ 使用 DeepSeek AI 深度分析
- ✅ 考虑季节、风格、购买习惯
- ✅ 返回 3 条精准建议

### 模式 2：本地规则（降级）
- 当 AI 不可用时自动启用
- 基于预定义规则生成建议
- 确保功能始终可用

### 如何判断使用的模式？

**查看控制台**（F12 打开开发者工具）：

```
✅ 使用 DeepSeek AI 建议  ← AI 模式
AI 总结: [AI 的总结]
```

或

```
⚠️ 使用本地规则生成建议（降级方案）  ← 本地模式
```

## 💡 使用技巧

### 1. 完善衣物信息
为了获得更准确的建议，请：
- 填写衣物品牌
- 添加风格标签（休闲、正式、运动等）
- 标注季节信息
- 记录购买时间

### 2. 定期记录穿搭
- 点击"记录穿搭"更新使用次数
- 帮助 AI 了解您的使用习惯
- 获得更个性化的建议

### 3. 创建更多搭配
- 搭配数量越多，AI 越了解您的风格
- 推荐至少创建 5-10 个常用搭配

### 4. 及时整理衣橱
- 标记破损或丢弃的衣物
- 定期清理不需要的衣物
- 保持衣橱数据的准确性

## ❓ 常见问题

### Q: 建议一直显示加载中？
**A**: 检查网络连接，或等待几秒钟。首次加载可能需要 10-15 秒。

### Q: 建议为什么是空的？
**A**: 可能是因为：
1. 衣橱中衣物太少（< 5 件）
2. 所有指标都很健康，没有需要改进的地方

### Q: 如何强制刷新建议？
**A**: 下拉页面，或等待 1 小时缓存过期后重新访问。

### Q: 建议不符合我的情况怎么办？
**A**: 
1. 检查衣物信息是否完整
2. 确保购买时间、使用次数等数据准确
3. 添加更多标签帮助 AI 理解您的风格

### Q: AI 模式和本地模式有什么区别？
**A**: 
- **AI 模式**：更智能、更个性化、考虑季节和风格
- **本地模式**：基础分析、快速响应、无需网络
